# Data_Merge_Tool

Scaffold directory for the formal repository maintenance tool.

## Planned responsibilities
1. Read RelayPackage
2. Perform precheck / conflict detection
3. Apply create / overwrite into `Data_Spilt` and `Picture`
4. Execute physical delete from `Data_Spilt` and `Picture`
5. Rebuild affected `Data_Merge` directories
6. Rebuild affected `INDEX.json` files
7. Output reports

## Current status
- Scaffold only
- No executable implementation included yet
